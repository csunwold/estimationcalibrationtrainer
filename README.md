# Estimation Calibration Trainer
An application that helps calibrate you as an estimator. 

## Dependencies
* Maven
* Scala 2.13

## How to build
```
$ mvn clean install
$ java -jar cli/target/cli-1.0-SNAPSHOT-jar-with-dependencies.jar
```
